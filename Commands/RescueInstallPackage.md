```
RescueInstallPackage
A minimal functionality InstallPackage that can be used when the original script is not working
Usage:   RescueInstallPackage <package_file|package_dir|package_url> [goboPrograms] [goboIndex]
Example: RescueInstallPackage /Data/Packages/Glibc--2.3.2--i686.tar.bz2 /Programs /System/Index

```
